Move


In this mode, the user can drag a point, a line, a circle and so on. To do this, place the mouse over a movable element, select the element by pressing the left mouse button, and drag it (with the button still pressed) to the desired position. Note that each time only one elements can be draged.

  1. Free Point. These points can be moved freely on the diagram.
  2. Point on Line. For the point which lies on a line, during the move this point slides along the line.
  3. Point on Circle. For the point that lies on a circle, during the move this slides along the circle.
  4. Line or Circle . For a line or a circle which only contains free point, all the points on the elements will be draged to new position.
  5. Other Elements. All elements in the diagram will be draged accordingly.
