• The Right Click Menu
    • Basic
      • Cancel Action
          • Cancel the current action and set the current action to "move".
      • Move
          • Cancel the current Action and set the current action to Move Mode.
      • Color
          • Change the current selected object's color.
      • Property
          • Popup the Property Dialog and let the user can change the property of the element.
    • On a Point
      • Freeze
          • Freeze the current point. That means the current point can not be draged. The position of this point can not be changed. This can be useful if the user have finished the drawing and don't want other people to change the diagram.
      • Trace
          • Trace the selected point. Please see the action Trace for detail.
      • Stop Trace
          • Stop to trace the point when a point is on tracing. This action make the program to stop tracing this point.
      • X Coordinate
          • Show the X Coordinate of the point.
      • Y Coordinate
          • Show the Y Coordinate of the point.
    • On Text
      • Edit Text
          • Popup a text window to edit this text.
      • Plain
          • Set the style of the font to be Plain.
      • Bold
          • Set the style of the font to be Bold.
      • Font Size
          • Choose the font size of the text.
    • On Line
      • Slope
          • Shows the slope of the selected line.
    • On Circle
      • Radius
          • Shows the radius of the selected circle.
      • Girth
          • Shows the girth of the selected circle.
      • Area
          • Shows the area of the selected circle.
