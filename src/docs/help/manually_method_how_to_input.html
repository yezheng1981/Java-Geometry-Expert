How to Input?

 

 

Step 1. Preparation.

To input the hypothese of the theorem, the user can just draw on the diagram pane and the corresponding text will be generated at the prove pane automatically.

 

Step 2. Input the Conclusion.

2.1 Use Button "Add" to popup a menu, select "Keywords --> To Prove" in the popuped menu.

2.2 Use Button "Append" to popup a menu, select "Assertion" or "Equation" in the popuped menu to input the conclusion to be proved.

 

Step 3. Proof Body

 


(1). Indicate the type of current input elements.

(2). The main input pane.

(3). Set the current proof is editable or not.

(4). Show the step bar.

(5). Button "OK".

(6). Button "Add". This button adds a new line to the current selected proof line .

(7). Button "Append". This button appends an elements to a proof line.

(8). Button "Child". This button creates a child to the current selected proof line.

(9). Button "Cancel". Cancel the current action.

 

 

The proof body can be divided into several types: Text, Keywords, Assertion, Objects, Draw, Equation,Rules.

1. Text. Select "Text" on the popuped menu, type in text in the input pane. Click OK button to comfirm the input.
