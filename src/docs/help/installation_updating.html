Installation and Updating.


JGEX is written purely in the Java. The install package of JGEX doesn't contain Java Virtual Machine, so the user need to download it from http://www.java.sun.com.

The JGEX software itself is available as a free download from the Woody homepage. Depending on your platform you either download an installer or the application itself.

 

Updating

To update JGEX, use the menu "Help -> Check for Update". JGEX will download update package automatically. When the package is downloaded completely, JGEX will be closed and the package will be unzipped. In this way, the user can update the software easily.
