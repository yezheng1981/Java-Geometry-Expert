A Quick Start

This is a quick start for JGEX.

Here are three examples which focus on three particular aspects.

1. Feuerbach's Theorem. This example focus on the drawing ability of JGEX. Through this example, the user will learn how to draw an diagram as well as make the diagram more beautiful.

2. The Nine Point Circle Theorem. This example introduces the automated theorem proving methods which are implemented in JGEX. Through this example, the user can lean how to draw the diagram as the input of the prover, how to set the conclusion and the how to generate and read the proof of the theorem.

3. Pythagoras' Theorem. This example introduce the manually created proof of Pythagoras's theorem.
