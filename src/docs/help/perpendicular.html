Add a Perpendicular Line


Click a line and click another point to construct a perpendicular line. The action is devided into three steps:

 

  1. Click on an existing line, the line will be selected.
  2. Move the pointer to another position, click again. This click will select or construct a point, See Add a Point.
  3. A parallel line is perpendicular. The line contains the selected point and is perpendicular to the selected line.
